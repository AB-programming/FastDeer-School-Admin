## 介绍 📦
快鹿（FastDeer）是一款面向青年群体和学生的综合性APP，整合了校园分帖、聊天、资源共享、学术资讯、校园活动发布与接收、校园就业信息发布与接收、志愿者服务信息发布与接收，平台反馈等功能，并配备了一套Web后台管理系统，以及各个校方的一套Web后台管理系统，旨在满足校园大众的多方面需求。

快鹿后台管理-学校 提供了对快鹿应用的校方后台管理服务，可进行活动、职位、志愿服务的发布与管理。
